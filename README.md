# CGI-proov-front

## Run locally

- Clone project repository
    - HTTPS `https://github.com/mrybrzy/cgi-proov-front.git`
    - SSH `git@github.com:mrybrzy/cgi-proov-front.git`
- Write `npm install` in the terminal
- Write `npm start` in the terminal
